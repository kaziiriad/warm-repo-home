

# Sultan Mahmud — Portfolio Website

A modern, single-page portfolio website showcasing your backend engineering expertise, projects, and professional profile.

## Design System
- **Primary color:** Deep Learning Purple (#9333FF)
- **Secondary color:** Tech Blue (#00A8E6)
- **Typography:** Inter for UI/body, JetBrains Mono for code snippets
- **Style:** Dark-themed with subtle card shadows, hover lift effects, and fade-in animations
- **Layout:** Responsive, mobile-first, max-width 1200px
